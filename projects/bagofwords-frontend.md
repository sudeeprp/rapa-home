# End-user frontend for Agentic data exploration

Agentic data exploration tools let you converse with data. Instead of using traditional methods of querying the database (like SQL), you can use natural language.

## Background

This project aims to build on an open-source tool: [bagofwords](https://github.com/bagofwords1/bagofwords)

The default UI provided in the tool is meant for data engineers. It shows details like the names of the tables and columns.

## Tasks

Your task is to make the UI suitable for end-users. Here are a few starter ideas:

- Hide text on the UI when it contains table names
- Replace the frontend with your own, having custom authentication and display
- Build own frontend + agent using the bagofwords MCP server

## Preparation

Setup your laptop with docker. Run the bagofwords tool with dummy data in postgresql. Demonstrate a few prompts in its frontend. This is the starting point for any of the above tasks.
